👨‍💻 Stuff I've worked on

👉 Sistem Ambilan Mahasiswa (SAM)

SAM started as an initiative with the Admissions Department to build an admission system that would act as a flexible and centralized top-of-the-funnel tool as UTMSPACE was diversifying into other products.

I learned a lot about product development and how critical understanding users is in building products.

👉 Sistem Maklumat Pelajar (SMP) v2

I rewrote SMP, an online academic management system for PSM (Program Separuh Masa) students. The old codebase was difficult to maintain and changes were hard to deploy. We implemented a CI/CD pipeline to automate builds and deployments.

The user interface was revamped to be easier to use and we also added a feature that encouraged students to provide feedback.

👉 Upgrade UTMSPACE’s payment gateway

I upgraded our payment gateway to conform to the changes introduced by FPX. We managed to save RM500,000 in development costs by reverse-engineering the existing system (which was outsourced) and rewriting it.

👉 Introduced a CI/CD Environment

I helped set up a central code repository and build-server. Since then it has helped us enhance collaborations and has made deployments much simpler and less error-prone.
